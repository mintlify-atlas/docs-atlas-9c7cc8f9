fCavEX includes many items from Minecraft Beta 1.7.3. All tools are currently unbreakable (durability system simplified to 8-bit).

## Tools

All tools come in five tiers: Wooden, Stone, Iron, Gold, and Diamond. Tools are currently unbreakable.

## Fuel Values

Some items can be used as furnace fuel:

| Item | Fuel Units |
|------|:----------:|
| Coal | 8 |
| Stick | 1 |
| Wooden Tools | 2 |
| Wooden Door | 2 |
| Bowl | 1 |
| Bow | 1 |

<Info>
  Furnaces hold up to 15 fuel units and consume 1 unit per smelted item. Smelting is instant.
</Info>
